DataStructers-Queue
===================

A simple example of the Queue

Queue
=====

FIFO (First in First Out)

queue[]     -> create queue
maxLength   -> max queue size
last        -> queue last/end

full()      -> is queue full
empty()     -> is queue empty
enqueue()   -> add to queue employee
dequeue()   -> remove the employee from the queue
showQueue() -> show queue
